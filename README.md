# SQL Interview Questions and Practice Queries

This repository contains commonly used SQL queries for Data Analyst, Business Intelligence and Data Engineering interviews.

I am creating this repository to keep SQL queries that are commonly used in real projects and technical interviews.

## Topics Covered
- Joins
- Window Functions
- CTE (Common Table Expressions)
- Subqueries
- Aggregations
- Case When
- Date Functions
- Running Total
- Top N Queries
- Duplicate Records

These SQL queries are useful for interview preparation and real-world data analysis scenarios.

More queries will be added regularly.
